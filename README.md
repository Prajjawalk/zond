## Installing

#### Ubuntu

**Dependencies:**

```
sudo apt -y install swig3.0 python3-dev build-essential cmake libhwloc-dev libboost-dev ninja-build pkg-config
```

**Install gcc-5**

```
sudo add-apt-repository ppa:ubuntu-toolchain-r/test
sudo apt-get update
sudo apt-get install gcc-5 g++-5
```
